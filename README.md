# AI Prompt Sharing App - README

This is a web application built with Next.js that allows users to share and explore AI-generated prompts. Users can create, edit, and delete their own prompts, as well as copy the prompted text for further use. The AI Prompt Sharing App aims to foster creativity, collaboration, and inspiration among users in a user-friendly and intuitive interface.

## Features

- **User Authentication**: Users can create accounts, log in, and log out securely to access the full functionality of the app.
- **Prompt Creation**: Users can create new prompts by providing a title and a description.
- **Prompt Editing**: Users have the ability to edit their own prompts, allowing them to refine and improve their content.
- **Prompt Deletion**: Users can delete their own prompts if they no longer wish to share them.
- **Prompt Listing**: Users can browse througha list of prompts shared by other users, gaining inspiration and insights from a diverse range of ideas.
- **Prompt Search**: Users can search for prompts based on keywords or specific criteria, making it easier to find prompts that align with their interests.
- **Prompt Copying**: Users can copy the prompted text with a single click, allowing them to use it as a starting point for their own creative projects.
- **User Profiles**: Each user has a profile page displaying their shared prompts and personal information.
- **Responsive Design**: The app is designed to be mobile-friendly, ensuring a seamless experience across different devices.

## Getting Started

To run the AI Prompt Sharing App locally, follow these steps:

1. Clone the repository from GitHub: `git clone https://github.com/your-username/ai-prompt-sharing-app.git`
2. Navigate to the project directory: `cd ai-prompt-sharing-app`
3. Install the dependencies: `npm install`
4. Set up the environment variables:
   - Create a `.env.local` file in the root directory.
   - Define the necessary environment variables in the file, such as database connection details, API keys, and session secrets.
5. Start the development server: `npm run dev`
6. Open your web browser and visit `http://localhost:3000` to access the app.
